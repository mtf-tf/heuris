
###wordpress用のデータベース作成（on MariaDB）

MariaDBにログイン
# mysql -u root -p

データベース作成
> CREATE DATABASE databasename;

データベースの確認
> show databases;

ユーザ作成と権限付加
> GRANT ALL PRIVILEGES ON wpdb.* TO "wpuser"@"localhost" IDENTIFIED BY "sakonndesu";

ユーザの確認
> select Host, User, Password from mysql.user;

切断
> exit
